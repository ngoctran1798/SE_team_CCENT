##Mạng căn bản  
1.Mạng là gì?  
- Là hệ thống gồm 2 hay nhiều máy tính liên kết để chia sẻ tài nguyên.  
- Được kết nối qua dây cáp,sống hồng ngoại ,vệ tinh.    
- Gồm 2 loại cơ bản:  
  - Local Area Network(còn được viết tẳ là LAN)    
  - Wide Area Network(còn được viết tắt là Wan)
